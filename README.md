# Administración de Sistemas en Linux 🐧

Este repositorio contiene una recopilación de los trabajos, prácticas y actividades que he realizado durante el módulo de **Administración de Sistemas en Linux**.

Los documentos están disponibles en formato PDF y muestran diferentes procedimientos relacionados con la instalación, configuración, mantenimiento y administración de sistemas GNU/Linux.

## 📚 Contenido

En este repositorio se pueden encontrar prácticas relacionadas con:

* Instalación y configuración de sistemas Linux.
* Administración de usuarios y grupos.
* Gestión de archivos, directorios y permisos.
* Configuración de redes.
* Gestión de procesos y servicios.
* Administración del almacenamiento.
* Uso de comandos y herramientas del sistema.
* Resolución y documentación de diferentes actividades prácticas.

## 📁 Organización

Cada práctica se encuentra guardada en un documento PDF independiente para facilitar su consulta y organización.

## 🎯 Objetivo

El objetivo de este repositorio es recopilar y mostrar los conocimientos y habilidades que he adquirido durante mi formación en administración de sistemas Linux.
